B. Bạn Hoàng
C. Bạn Đức

Lí thuyết:
Các số có hai chữ số (từ 21 đến 40 )

Từ hình vẽ, ta có:

Bạn Hoàng mang áo số 26 , số 26 có cách đọc là: hai mươi sáu.

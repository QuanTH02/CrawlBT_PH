Câu 11 : [VD]

Bạn hãy chọn đáp án đúng.

Để về được đến chuồng, chú bò phải đi qua các số:

A. Bảy mươi hai, bảy mươi chín, tám mươi tư, chín mươi ba, tám mươi tám. B. Bảy mươi hai, bảy mươi tư, tám mươi ba, tám mươi tư. C. Bảy mươi hai, bảy mươi chín, tám mươi tư, chín mươi hai.
D. Bảy mươi hai, bảy mươi chín, tám mươi tư, chín mươi ba.
Bước 1:

Lí thuyết:
Các số có hai chữ số ( 71 đến 99 ).

Để về được đến chuồng, chú bò phải đi qua các số: 72 ; 79 ; 84 ; 93.

Do đó, chú bò phải đi qua các số: Bảy mươi hai, bảy mươi chín, tám mươi tư, chín mươi ba.
Vậy đáp án đúng là: Bảy mươi hai, bảy mươi chín, tám mươi tư, chín mươi ba.

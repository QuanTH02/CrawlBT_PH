Câu 1 : [TH]

Bạn hãy chọn đáp án đúng
Một hình chữ nhật có chiều dài bằng 24 c m . Biết chiều dài gấp 3 lần chiều rộng. Tìm diện tích của hình chữ nhật đó?

A. 8cm2 B. 72cm2 C. 576cm2
D. 192cm2
Bước 1:

Lý thuyết: Diện tích hình chữ nhật = chiều dài × Chiều rộng (Cùng đơn vị đo)

Chiều rộng hình chữ nhật là: 24 : 3 = 8 ( c m )
Diện tích hình chữ nhật là: 8 × 24 = 192 ( c m 2 ) \
Đáp án: 192 c m 2

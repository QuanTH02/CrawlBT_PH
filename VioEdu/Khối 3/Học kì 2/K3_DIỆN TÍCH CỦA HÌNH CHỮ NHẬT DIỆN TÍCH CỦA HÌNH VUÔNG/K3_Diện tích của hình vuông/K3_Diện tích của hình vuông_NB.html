16
64

25;
>.

Nhận xét nào dưới đây đúng?

A. a<65cm2
B. a>65cm2 C. a=65cm2
Bước 1:

Lý thuyết:
Muốn tính diện tích hình vuông ta lấy độ dài một cạnh nhân với chính nó.

Diện tích của hình vuông là: 8 × 8 = 64 ( c m 2 ) .
Vì 64 c m 2 < 65 c m 2 nên a < 65 c m 2 .
Đáp án:
a < 65 c m 2

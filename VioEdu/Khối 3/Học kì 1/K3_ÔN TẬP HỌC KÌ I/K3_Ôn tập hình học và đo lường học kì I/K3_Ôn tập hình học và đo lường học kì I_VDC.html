Câu 1 : [VDC]

Bạn hãy chọn đáp án đúng.
Cho hai hình tròn, hình tròn tâm O có bán kính bằng 10 c m , hình tròn tâm I có đường kính bằng 10 c m và đoạn thẳng O I có độ dài bằng 12 c m . Độ dài đoạn thẳng C B bằng bao nhiêu?

A. 3cm
B. 5cm C. 2cm D. 6cm
Bước 1:

Lý thuyết:
Hình tròn, tâm, bán kính, đường kính.

Vì C I = C B + B I nên:
O B + I C = O B + ( C B + B I ) = O B + B I + C B = ( O B + B I ) + C B
Vì O I = O B + B I nên O B + I C = O I + C B .
Vậy C B = O B + I C − O I .
O B là bán kính hình tròn tâm O nên độ dài O B là 10 c m .
I C là bán kính hình tròn tâm I nên độ dài I C là: 10 : 2 = 5 ( c m ) .
Độ dài C B là: 10 + 5 − 12 = 3 ( c m ) .
Đáp án:
3 c m

Bán kính hình tròn tâm O dài hơn bán kính hình tròn tâm I là 4 c m nên độ dài đoạn O I là 4 c m .
B là trung điểm của đoạn thẳng O I nên độ dài O B là: 4 : 2 = 2 ( c m ) .
Đáp án:

Câu 3 : [VDC]

Hình tròn tâm I

Hình tròn tâm P

Hình tròn tâm O

Lý thuyết:
Hình tròn, tâm, bán kính, đường kính.

Hình tròn tâm I , hình tròn tâm P , hình tròn tâm O .

A. 86−(98−21)
B. 72:(2×4)
C. 142−42:2 D. (142−42):2
Bước 1:

Lý thuyết:
Khi tính giá trị của biểu thức có dấu ngoặc ( ) thì trước tiên ta thực hiện các phép tính trong ngoặc.

Ta có: 86 − ( 98 − 21 ) = 86 − 77 = 9
72 : ( 2 × 4 ) = 72 : 8 = 9
142 − 42 : 2 = 142 − 21 = 121
( 142 − 42 ) : 2 = 100 : 2 = 50
Vậy các biểu thức có giá trị bằng 9 là: 86 − ( 98 − 21 ) ; 72 : ( 2 × 4 )
Đáp án:
86 − ( 98 − 21 )
72 : ( 2 × 4 )

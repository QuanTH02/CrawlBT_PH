Câu 1 : [VDC]

Chọn đáp án đúng.

Tính độ dài đường gấp khúc A B C D , biết độ dài đường gấp khúc A B C = 22 c m , độ dài đường gấp khúc B C D = 33 c m .

A. 50cm
B. 55cm C. 45cm D. 65cm
Bước 1:

Lý thuyết:
Độ dài đường gấp khúc bằng tổng độ dài các đoạn thẳng của nó.

Ta có độ dài đường gấp khúc A B C là 22 c m nên A B dài: 22 − 5 = 17 c m
Độ dài đường gấp khúc B C D là 33 c m nên C D dài: 33 − 5 = 28 c m
Độ dài đường gấp khúc A B C D là: A B + B C + C D = 17 + 5 + 28 = 50 c m .
Đáp án đúng: 50 c m .

Ống dây thép ngắn hơn ống dây đồng số xăng-ti-mét là:

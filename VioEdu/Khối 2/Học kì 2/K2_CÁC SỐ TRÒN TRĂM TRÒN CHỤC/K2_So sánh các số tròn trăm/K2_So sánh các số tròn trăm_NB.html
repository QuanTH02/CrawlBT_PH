Câu 1 : [NB]

Bạn hãy chọn các đáp án đúng.
(Có thể chọn nhiều hơn một đáp án)
Trong các khẳng định dưới đây, khẳng định nào đúng?

A. 100<200.
B. 300=500. C. 400>800.
D. 700=700.
Bước 1:

Lý thuyết:
Cách so sánh số tròn trăm: Số nào có chữ số hàng trăm càng lớn thì có giá trị càng lớn.

Vì 1 < 2 nên 100 < 200.
Vì 3 < 5 nên 300 < 500.
Vì 4 < 8 nên 400 < 800.
Vì 7 = 7 nên 700 = 700.
Vậy 100 < 200 và 700 = 700 là đúng.
Đáp án:
100 < 200.
700 = 700.

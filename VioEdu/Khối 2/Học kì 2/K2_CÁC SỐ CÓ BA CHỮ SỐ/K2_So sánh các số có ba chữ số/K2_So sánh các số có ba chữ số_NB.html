+ Đầu tiên, ta so sánh chữ số hàng trăm: Số nào có chữ số hàng trăm lớn hơn thì số đó lớn hơn.

+ Nếu cùng chữ số hàng trăm thì so sánh chữ số hàng chục: Số nào có chữ số hàng chục lớn hơn thì số đó lớn hơn.

+ Nếu cùng chữ số hàng trăm và chữ số hàng chục thì so sánh chữ số hàng đơn vị: Số nào có chữ số hàng đơn vị lớn hơn thì số đó lớn hơn.
